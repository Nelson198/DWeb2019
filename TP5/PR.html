Abstract:

No quinto trabalho prático da unidade curricular de Desenvolvimento de Aplicações WEB foi requirido a realização de um exercício que aborda o tema indicado acima.

Este exercício consistia na criação de um servidor, escrito em JavaScript / Node.js, que permitia gerir um conjunto de várias tarefas. Como tal, este gestor de tarefas é composto por duas funcionalidades: submissão de tarefas e visualização de todas as tarefas submetidas até ao momento.

Por forma a agilizar o processo de codificação das páginas Web intrínsecas a este projeto, foi utilizada uma biblioteca JavaScript denominada por Pug.js. Esta biblioteca possui recursos poderosos, como condições, ciclos, inclusões e mixins, com os quais podemos realizar, de forma simplificada, a renderização de código HTML. De salientar também que o Pug suporta nativamente JavaScript.

Nota: foi utilizada a stylesheet disponibilizada pela página W3 Schools (w3.css) para a maioria das páginas Web geradas neste trabalho prático.

Deliverables:

  • Ficheiro JSON relativo ao conjunto de tarefas submetidas
  • Servidor responsável pela gestão de tarefas
  • Ficheiro Pug que especifica a estrutura base da página Web associado ao gestor de tarefas
  • Ficheiro Pug que codifica o corpo da página Web relativo ao gestor de tarefas
  • Ficheiro Pug que traduz a página Web relativa a um determinado erro
