Abstract:

Relativamente ao 1º exercício do teste é pedido a construção de uma API de dados. Por forma a fazer a persistência da base de dados, foi convertido o ficheiro XML para o formato JSON através deste website . Consequentemente, esta base de dados com o nome obras foi importada para o MongoDB através do comando mongoimport com a opção --jsonArray . A porta do servidor relativa à API é a porta com nº 1000.

Relativamente ao 2º exercício do teste é pedido a obtenção de alguns resultados acerca da API de dados da plataforma CLAV . As respostas a estas perguntas encontram-se disponíveis no ficheiro queries.txt . Quanto à implementação da interface pedida foram criadas 2 páginas Web , uma relativa à listagem das entidades e outra relativa à informação associada à mesma. Por forma a aceder a esta última informação é necessário clicar no ícone da lupa presente da página da listagem das entidades. Por forma a aceder à API de dados da plataforma CLAV foi utilizado a biblioteca axios . A porta do servidor relativa à interface é a porta com nº 2000.

De maneira a estabelecer a comunicação entre o servidor desenvolvido e o MongoDB foi utilizada a biblioteca Mongoose . Esta é uma biblioteca ODM (Object Data Modeling) para MongoDB e Node.js que permite gerir relacionamentos entre dados, fornecer validação de esquema e, ainda, traduzir objetos em código e, posteriormente, a sua representação no MongoDB .
