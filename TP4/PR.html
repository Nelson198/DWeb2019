Abstract:

No quarto trabalho prático da unidade curricular de Desenvolvimento de Aplicações WEB foi pedido a realização de dois exercícios que abordam o tema indicado acima.

Em primeiro lugar foi requirido a criação de um transformador em XSL para converter o ficheiro relativo aos arqueossítios (também disponível na secção seguinte) em vários documentos XML, um para cada um dos arqueossítios individuais (conteúdo presente entre as tags ARQELEM);

No segundo e último exercício deste trabalho prático foi pedido a criação de um servidor de ficheiros, escrito em JavaScript / Node.js, que permite aceder a um determinado documento especificado na query string do URL.

Deliverables:

  • (arq.xml) : Arqueossítios - XML
  • (arq2html.xsl) : Ficheiro XSL responsável por transformar cada arqueossítio (XML) numa página HTML
  • (xml-split.xsl) : Ficheiro XSL que cria um documento XML por cada arqueossítio presente em arq.xml
  • Pasta que contém os ficheiros XML de cada arqueossítio e ainda alguns ícones ou imagens
  • Página Web que é apresentada quando o arqueossítio específicado não existe
  • Servidor de ficheiros
