Abstract:

No terceiro trabalho prático da unidade curricular de Desenvolvimento de Aplicações WEB foi pedido a realização de dois exercícios que abordam as transformações de documentos XML.

Numa primeira instância era requerido a transformação do ficheiro PR.xml, que respeita as regras definidas no ficheiro PR.dtd que foi desenvolvido nas aulas e que foi disponibilizado pelo docente, para um outro com o formato .html. Para executar esta tarefa é preciso criar um documento .xsl que permite aplicar esta mesma transformação. Com isto, obtém-se como resultado uma página Web com o conteúdo presente no documento XML em causa.

Por fim, no segundo e último exercício é solicitado igualmente a criação de um ficheiro com extensão .xsl, isto é o documento arq.xsl, que possibilita a transformação do ficheiro relativo aos arqueossítios (já presente no trabalho prático anterior) num Website. Para tal, de forma a visualizar uma listagem de todos os arqueossítios presentes, é concebido um índice que é posteriormente ilustrado numa página Web específica, ou seja, em index.html. Para além disto, é também elaborada uma página Web para cada arqueossítio presente no documento XML em questão, exibindo-se informação relacionada com o mesmo.
