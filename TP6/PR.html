Abstract:

No sexto trabalho prático da unidade curricular de Desenvolvimento de Aplicações WEB foi requirido a realização de dois exercícios que abordam o tema indicado acima.

No primeiro exercício era pedido a transformação do ficheiro arq-son-EVO.xml para um documento JSON que, posteriomente, irá servir como base de dados para a aplicação requerida. Uma vez que o era permitido, optei por fazer esta tarefa na seguinte página Web.

O segundo exercício consistia na criação de um servidor, escrito em JavaScript / Node.js, que permitia gerir a base de dados em causa, isto é, o ficheiro JSON arq-son-EVO.json. Como tal, este gestor de tarefas é composto por quatro funcionalidades: adição, remoção, edição e consulta de registos.

Tal como é indicado no título desta página, foi utilizada a framework Express para criação do servidor pretendido. A partir desta foram criadas várias views (que permitem renderizar várias páginas HTML) e também alguns ficheiros Javascript (que tratam de executar algumas ações intrínsecas ao funcionamento do servidor).

Por forma a agilizar o processo de codificação da maioria das páginas Web associadas a este projeto, foi utilizada uma biblioteca JavaScript denominada por Pug.js. Esta biblioteca possui recursos poderosos, como condições, ciclos, inclusões e mixins, com os quais podemos realizar, de forma simplificada, a renderização de código HTML. De salientar também que o Pug suporta nativamente JavaScript.

Nota: foi utilizada a stylesheet disponibilizada pela página W3 Schools ( w3.css) para a maioria das páginas Web geradas neste trabalho prático.
